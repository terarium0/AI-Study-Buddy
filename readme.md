# AI Study Buddy

AI assistant untuk membantu siswa memahami materi PDF secara otomatis.

## Features
- PDF summarizer
- Quiz generator
- AI mind map
- Simple explanation

## Tech Stack
- Gemini API
- Streamlit
- Graphviz
- PyMuPDF

## How To Run
pip install -r requirements.txt
streamlit run app.py
